# MailBox UI

シンプルなメールアプリ風UIをHTML/CSS/JavaScriptで作ったデモプロジェクトです。

GitHub Pagesでそのまま公開できる構成になっています。

---

## 🚀 デモ

GitHub Pagesで公開後、ここにURLを追加してください：



[https://qianyu-08.github.io/mailbox-ui/](https://qianyu-08.github.io/mailbox-ui/)


---

## 📦 特徴

- 受信箱 / 送信済み / 下書き / ゴミ箱のUI
- メール一覧 + プレビュー表示
- フィルタ・検索機能
- 新規メール作成モーダル
- レスポンシブ対応（PC / タブレット / スマホ）

---

## 🧠 技術

- HTML5
- CSS3（Glassmorphism風デザイン）
- Vanilla JavaScript（フレームワークなし）

---

## 📁 構成

```

index.html
README.md

```

---

## 🛠 使い方

1. リポジトリをクローン
2. `index.html` をブラウザで開く
3. もしくは GitHub Pages を有効化

---

## 🌐 GitHub Pages 公開手順

1. Settings を開く
2. Pages セクションへ移動
3. Source を `main` / root に設定
4. Save
5. 表示されたURLにアクセス
